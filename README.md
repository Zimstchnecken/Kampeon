# Kampeon

Kampeon is an adventure storytelling game developed using GDScript in the Godot Engine.
This project was created as part of a Software Engineering course.

## Features

* Story-based gameplay
* Interactive elements
* Developed with Godot and GDScript

## Getting Started

To run the project:

1. Install the Godot Engine: [https://godotengine.org/](https://godotengine.org/)
2. Open the project folder in Godot
3. Run the main scene
